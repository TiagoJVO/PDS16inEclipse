Projeto

O projeto PDS16inEcplise consiste no desenvolvimento de uma ferramenta que visa facilitar a escrita de programas para o processador PDS16[[1]] usando a sua linguagem assembly. Este plug-in é, essencialmente, um editor de texto que integra funcionalidades para fazer uma verificação da semântica e da sintaxe em tempo de escrita de código. O desenvolvimento desta ferramenta é baseado na framework Xtext integrada no Ambiente Integrado de Desenvolvimento (IDE) Ecplise.

Assembly PDS16

A arquitetura PDS16[[2]] foi desenvolvida no Instituto Superior de Engenharia de Lisboa (ISEL), em 2008, com o objetivo de suportar não só uma mais fácil compreensão mas também o ensino experimental dos conceitos básicos subjacentes ao tema “Arquitetura de Computadores”. Esta arquitetura a 16 bits adota a mesma filosofia das máquinas do tipo Reduced Instruction Set Computer (RISC), oferecendo o seu Instruction Set Architecture (ISA) ao programador 6 registos de uso geral e cerca de 40 instruções distintas, organizadas em três classes: 6 instruções para controlo do fluxo de execução, 18 instruções de processamento de dados e 12 instruções de transferência de dados, (ver [[2]]).

Processo de Instalação

Para fazer uso correto do editor de texto PDS16inEclipse é necessário instalar o plug-in e definir uma variável de ambiente do sistema operativo para utilização do assembler DASM [[3]]. Plug-in PDS16inEclipse pode ser instalado no IDE Eclipse de duas formas: 1) fazendo o download de um ficheiro ZIP ou instalar diretamente usando o link. Para qualquer dos casos, para instalar o plug-in é necessário seguir os seguintes passos:

  • Definir uma variável de ambiente do sistema operativo com o nome "DASM_PATH" que identifica a path para o assembler DASM;
  • No IDE Eclipse, efetuar os seguintes passos:
    • Ir à tab "Help" -> "Install New Software";
    • Clicar em "Add" e no campo "Location" colocar o endereço web do plug-in ou em alternativa a path para a diretoria para onde foi descompactado o ficheiro zip;
    • Selecionar o software "Pds16inEclipse" e prosseguir a instalação.
